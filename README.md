# 📦 Sell Through Rate (STR) Calculator

## 🔍 What is STR?
Sell Through Rate = Units Sold / Units Available

## 🚀 Features
- Calculates STR for each product
- Shows percentage STR
- Identifies best-selling product

## 🛠 Tech Used
- Python
- Pandas

## 📈 Example Output
Product_A → STR = 70%
Product_C → Best Product

## 💡 Use Case
Helps businesses track inventory performance and optimize sales strategy.
